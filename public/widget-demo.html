Embeddable Chat Widget

Add AI-powered document search to any website with a single <script> tag. The floating chat button (bottom-right ↘) is live — try asking it anything!

Quick Embed

<script
  src="http://localhost:8000/widget.js"
  data-widget-key="YOUR_WIDGET_PUBLIC_KEY"
  data-position="bottom-right"
></script>
